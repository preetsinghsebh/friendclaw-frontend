# BuddyClaw Frontend

The official web interface for **BuddyClaw**, the world's first AI companion ecosystem that doesn't sound like a bot.

## ✨ Features

- **Real-time Chat**: Interaction with AI companions like Ziva, Liam, and more.
- **Vibes Dashboard**: Daily affirmations, horoscopes, and reflections.
- **Founder's Circle**: Exclusive access and status for early supporters.
- **Responsive Design**: Premium dark-mode UI optimized for all devices.
- **Supabase Auth**: Secure and seamless user authentication.

## 🛠️ Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS & Vanilla CSS
- **Animations**: Framer Motion
- **3D Elements**: Three.js & React Three Fiber
- **Authentication**: Supabase
- **Database**: Mongoose (for shared data)

## 🚀 Getting Started

1.  Navigate to the web directory:
    ```bash
    cd web
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up your environment variables in `.env.local`.
4.  Run the development server:
    ```bash
    npm run dev
    ```

## 🌐 Deployment

The frontend is optimized for deployment on Vercel or as a static export, connecting to the BuddyClaw microservices architecture.
